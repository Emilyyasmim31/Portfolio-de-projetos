< Habilidades />

HTML

Linguagem de marcação para estruturar o conteúdo web. É a base de qualquer site, definindo elementos como títulos, parágrafos, imagens e links.

CSS

Linguagem de estilização que controla a aparência visual dos elementos HTML. Permite criar layouts responsivos, animações e designs personalizados.

JavaScript

Linguagem de programação que adiciona interatividade às páginas web. Permite criar funcionalidades dinâmicas, manipular dados e responder a eventos do usuário.

Scratch

Linguagem de programação visual ideal para iniciantes. Permite criar jogos, animações e histórias interativas através de blocos de código que se encaixam como peças de quebra-cabeça.
